## 2.4. 赋值

使用赋值语句可以更新一个变量的值, 最简单的赋值语句是将要被赋值的变量放在 `=` 的左边, 新值的表达式放在 `=` 右边.

```Go
x = 1                       // 命令变量的赋值
*p = true                   // 通过指针间接赋值
person.name = "bob"         // 结构体字段赋值
count[x] = count[x] * scale // 数组, 切片 或 字典的 元素赋值 
```

特定的赋值语句和二元算术复合操作有一个简洁形式, 例如上面最后的语句可以重写为:

```Go
count[x] *= scale 
```

这样可以省去对变量表达式的重复计算.

数值变量也可以支持 `++` 递增和 `--` 递减语句:

```Go
v := 1 
v++    // 等价方式 v = v + 1; v 变成 2 
v--    // 等价方式 v = v - 1; v 变成 1 
```
