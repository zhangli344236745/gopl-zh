### 从源文件构建:

先安装 Go语言环境, git 工具 和 GitBook 命令行工具(`npm install gitbook-cli -g` 命令).

1. 运行 `go get github.com/golang-china/gopl-zh`, 获取 源文件
2. 运行 `go generate github.com/golang-china/gopl-zh`, 生成 `_book` 目录
3. 打开 `_book/index.html` 文件
